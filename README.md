# Wedding Website

A SvelteKit site styled with Tailwind CSS.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Structure

- `src/routes/+page.svelte` — landing page content
- `src/routes/+layout.svelte` — page layout and header
- `src/app.css` — Tailwind CSS entrypoint
